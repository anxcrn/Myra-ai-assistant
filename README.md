# MYRA AI 🚀

MYRA AI is a futuristic enterprise-grade Android AI assistant inspired by Jarvis, ChatGPT Voice, and next-generation autonomous AI systems.

## Features

- Real-time AI conversations
- Telugu + English multilingual AI
- Emotional AI system
- Floating AI orb overlay
- Wake word detection
- Accessibility automation
- Vision AI
- Autonomous AI agents
- Long-term memory engine
- FastAPI backend
- WebSocket streaming
- Material 3 cyberpunk UI

## Android Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Hilt Dependency Injection
- Room Database
- Coroutines + Flow
- Retrofit
- WebSockets

## Backend Stack

- FastAPI
- AsyncIO
- PostgreSQL
- Redis
- JWT Authentication
- Celery
